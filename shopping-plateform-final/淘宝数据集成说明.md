## 扩展建议

### 添加更多商品
编辑 `HomePage.ets` 中的 `taobaoProducts` 数组，添加更多商品数据

### 动态加载数据
可以从 `rawfile/taobao_products.json` 动态加载商品数据：
```typescript
import { resourceManager } from '@kit.Localization';

// 读取JSON文件
let resMgr = getContext().resourceManager;
let fileData = await resMgr.getRawFileContent('taobao_products.json');
let products = JSON.parse(new TextDecoder().decodeToString(fileData));
```

### 添加商品详情页
创建商品详情页，显示完整商品信息：
- 多张商品图片轮播
- 详细价格信息
- 店铺信息
- 发货地信息
- 商品详情描述

## 注意事项
- 图片URL来自淘宝CDN，需要网络权限
- 建议在 `module.json5` 中添加网络权限：
```json
"requestPermissions": [
  {
    "name": "ohos.permission.INTERNET"
  }
]
```
